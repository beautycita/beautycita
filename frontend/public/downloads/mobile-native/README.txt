BeautyCita Mobile Native App v1.2.0
====================================

Built with React Native + TypeScript
Platform: iOS & Android
Build Date: October 29, 2025

DOWNLOAD FILES:
- BeautyCita-v1.2.0.apk (Android - ~45 MB)
- BeautyCita-v1.2.0.ipa (iOS - ~38 MB)
- RELEASE_NOTES_v1.2.0.md (Full release notes)

INSTALLATION:

Android:
1. Download BeautyCita-v1.2.0.apk
2. Enable "Install from Unknown Sources" in Settings > Security
3. Tap the APK file to install
4. Open BeautyCita app

iOS:
1. Download BeautyCita-v1.2.0.ipa
2. Install via:
   - Xcode (for developers)
   - AltStore (for sideloading)
   - TestFlight (for beta testing)
3. Trust the developer certificate in Settings
4. Open BeautyCita app

FEATURES:
- Biometric authentication (Face ID, Touch ID, Fingerprint)
- Real-time chat with Socket.IO
- Google Maps integration
- Camera & photo gallery access
- Native Stripe payments (Apple Pay, Google Pay)
- Push notifications via OneSignal
- Full offline support
- Dark mode
